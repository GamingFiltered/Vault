# THE VAULT 🗝️

A modern, high-converting deal showcase platform with an interactive lock-dial daily unlocker, personalized matchmaker quiz, and built-in click/revenue analytics dashboard.

## Features
- **Interactive Dial Spin**: Gamified "Daily Unlock" spinner with physical dial animations.
- **Product Matchmaker**: Multi-step preference quiz to recommend personalized deals.
- **Admin Dashboard**: Live click tracking, customizable commission metrics, and real-time inventory management.
- **Local Persistence**: Dynamic fallback storage using LocalStorage.

## Deployment
Host for free on **GitHub Pages**, **Vercel**, or **Netlify**.
